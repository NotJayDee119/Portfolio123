# Jay Dee Magkidong Portfolio

A mobile-responsive professional portfolio website built with plain HTML, CSS, and JavaScript.

## Included Sections

- Landing page with name, professional title, and hero image
- About Me (background, education, mission and vision, goals and motivation, key achievements)
- Core Skills and Services (technical, soft skills, software proficiencies, specialized services)
- Experience and Projects
- Contact section with direct links and message form
- Download Resume (PDF) button

## Local Preview

Open `index.html` in your browser, or use VS Code Live Server.

## Deploy To GitHub Pages

1. Create a new repository and push this project.
2. In GitHub, open `Settings > Pages`.
3. Under `Build and deployment`, set:
	- Source: `Deploy from a branch`
	- Branch: `main` and folder `/ (root)`
4. Save and wait for deployment.
5. Your site URL will look like:
	`https://<your-username>.github.io/<repo-name>/`

## Deploy To Vercel

1. Go to https://vercel.com and import your GitHub repository.
2. Framework preset: `Other`.
3. Build command: leave empty.
4. Output directory: leave empty.
5. Click `Deploy`.

Vercel will provide a public URL after deployment.
